Daze.js

Node.js Web Framework

Get Start

# Easy to use

# install CLI tools
$ npm install -g @dazejs/cli

# create Daze.js project
$ daze create daze-example

# goto project dir
$ cd daze-example

# start service, default port is 8000
$ npm start
