Daze.js

基于 Node.js 的 Web 服务端框架

现在开始

# 简单易用

# 安装 CLI 工具
$ npm install -g @dazejs/cli@next

# 创建 Daze.js 工程
$ daze create daze-example --next

# 进入工程 目录
$ cd daze-example

# 开启服务，默认 8080 端口
$ npm start
